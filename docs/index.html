Portfolio Optimization Reports

Static HTML hosted via GitHub Pages.

How to add reports

  • Export Plotly charts/notebooks to HTML and place them in docs/.
  • Commit + push to main.
  • In GitHub: Settings → Pages → Deploy from branch: main, folder: /docs.

Reports

Add links here as you create HTML files.

Notes

  • If an HTML file includes large embedded Plotly JS, consider exporting with Plotly JS via CDN to keep file sizes smaller.
  • Keep any referenced assets (images, JS, CSS) under docs/ and use relative paths.
